## 6. Z 字形变换

[原题链接](https://leetcode-cn.com/problems/zigzag-conversion/)

### 思路

模拟过程：

1. 从第一行开始垂直向下走，即走从 `1` 走到 `numRows`
2. 到了最后一行后垂直向上走，从 `numRows` 再走回 `1`

循环这个过程。设置一个行走方向值 `forward`，当到达边界值时回头即可。

```python
class Solution(object):
    def convert(self, s, numRows):
        """
        :type s: str
        :type numRows: int
        :rtype: str
        """
        d = dict()
        index = 0
        forward = 1
        for c in s:
            if index not in d:
                d[index] = ""
            d[index] += c

            if index == numRows - 1:
                forward = -1
            if index == 0:
                forward = 1
            
            index += forward
        
        res = ""
        for index in d:
            res += d[index]
        return res
```
